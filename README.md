# Sahil's Content Engine

> My standing PR team as code. It reads everything I've built — my repos, my students' wins — banks the stories, learns my voice, and turns them into scroll-stopping video scripts with a hard CTA. Every run makes it richer. This is the one page I keep enriching.

## What's here

- **`CONTENT-SPRINT.md`** → start here. The shoot checklist, ranked by leverage, in priority order.
- **`scripts/`** → dual-cut scripts (Reel + LinkedIn), one hard CTA each:
  - `1-pdf/` — $100 PDF · stepaheadco.com/ship (global scale)
  - `2-webinar/` — bi-weekly live · 100 AED (recurring income)
  - `3-automation/` — automation / build-in-public (business owners)
  - `4-bigticket/` — company-transformation consulting
- **`narrative-bank.md`** → every win story, file-backed, append-only. The compounding asset.
- **`voice-profile.md`** → how I talk, so every script sounds like me.
- **`higgsfield/`** → the AI-video channel (my Soul ID + my voice), with a spend-gate and a one-shot test.
- **`captures/`** → I drop a new win here (a line, a screenshot note); the engine banks it on the next run.
- **`CLAUDE.md`** → the engine brain: how to run it and how it compounds.

## How I use it

**Today (shoot):** open `CONTENT-SPRINT.md`, film top-down. Each script tells me the hook, what to show on screen beat-by-beat, and the exact CTA. PDF angle first.

**Ongoing (compound):** when I ship something new or a student wins, I drop a note in `captures/` and say **"run the engine."** It mines the new win, appends to the bank, writes fresh scripts, and logs the run. The bank only grows.

**AI reels:** for cinematic/flex concepts, the `higgsfield/` channel renders them with my Soul ID. It always locks the look on cheap stills and waits for my sign-off before spending video credits.

## Priority order (fixed)

**PDF → Webinar → Automation → Big-ticket.** Scale globally first, lock in recurring income second, build authority third, land big-ticket fourth.
