**The throughline every script hits (kept tight so it doesn't rush):**
1. *"AI tutorials are overwhelming"* — the honest pain.
2. *A real system Sahil built* + his line: **Claude Code / agentic workflows are the future of building.**
3. *One starter prompt that coaches you step by step — no code.*
4. *"I built my own — it changed my life. More done, far less time."*
5. **CTA (no website name): click the link below, download 10+ business-automation starter prompts that make you full-stack with zero coding — first call with Sahil free.**

**No URL spoken or on-screen** — the avatar always says "click the link below." (You attach your stepaheadco.com/ship link to the post yourself.)

**Pacing note:** these are trimmed ~40% so a 15s clip doesn't sound rushed. If it still races, set Duration to **20s** — that's the comfortable read.

**Keep it honest:** the avatar voices the *real* mechanism (one starter prompt, coach + verify), Sahil's *real* builds, and Sahil's *real* free-first-call offer. No invented metrics.

**Spend-gate:** ~150–190 credits per generation. Generate **one**, watch it, fix, then the rest.
